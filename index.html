A powerful tool that generates tailored interview questions and answers based on a specific job role. It helps candidates prepare effectively by providing a comprehensive set of potential questions.

  • Frontend: React.js, Tailwind CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • AI Integration: OpenAI API (GPT-4) / LangChain
Features:
  • Enter any job role (e.g., “Python Developer”) to get started.
  • Scrapes job listings or uses keyword input for question generation.
  • Generates a variety of question types: Multiple Choice, Technical Q&A, and HR Questions.
  • Allows users to export the generated Q&A as a PDF for offline study.
  • Includes an admin panel for managing question templates and content.
